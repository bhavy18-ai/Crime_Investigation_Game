# Crime_Investigation_Game
# Overview
The Crime Investigation Game is a point-and-click mystery adventure developed using Python and Pygame. Designed as a B.Tech 1st-year project, it allows players to explore interactive crime scenes, gather clues, interrogate suspects, and solve cases in real time.
# Why This Project is Useful
Educational Value: Enhances critical thinking, problem-solving, and analytical skills.

Skill Development: Promotes logical reasoning and attention to detail.

Entertainment: Offers an engaging mystery-solving experience with immersive gameplay.

Training Potential: Adaptable for law enforcement training simulations.

Accessibility: Open-source tool for learning programming through gamification.
# Features
 Interactive Crime Scene Exploration
 
 Multiple Cases, Clues, and Suspects
 
 Real-time Gameplay Mechanics (timer & scoring system)
 
 Clue Validation & Suspect Interrogation
 
 Point-and-Click Interface with UI Components
 
 Visual Effects (pulsing glow for clues)
 
 Win/Lose Conditions & Complete Game Loop
 # Technical Implementation
Core Technologies: Python, Pygame

Architecture: Event-driven design with GameState class & scene management

Gameplay Mechanics: Scene navigation, clue validation, suspect profiles

Completed Features:

5 clues implemented,
3 suspects with profiles,
Timer system, scoring system &
Interactive UI
